# Physical AI & Humanoid Robotics: Bridging the Digital Brain and the Physical Body

This repository contains the documentation for the "Physical AI & Humanoid Robotics" course, built with Docusaurus.

## Overview

This course provides a comprehensive introduction to Physical AI and humanoid robotics, organized into weekly modules that build upon each other to provide a complete understanding of embodied artificial intelligence.

## Topics Covered

- Introduction to Physical AI and why it matters
- ROS 2 fundamentals for robotic communication
- Digital twin technology and simulation
- AI integration in robotic systems
- Vision-Language-Action systems
- Hardware setup and lab requirements

## Installation

```bash
# Clone the repository
git clone https://github.com/your-organization/physical-ai-humanoid-robotics.git

# Navigate to the project directory
cd physical-ai-humanoid-robotics

# Install dependencies
npm install

# Start the development server
npm run start
```

## Building for Production

```bash
npm run build
```

The built site will be in the `build/` directory and can be deployed to any static hosting service.
